# 📊 Financial Performance Dashboard (Power BI)

## 🔍 Project Overview

This project analyzes financial performance across different countries, segments, and products using Power BI.

The dashboard provides insights into key business metrics such as:

* Total Sales
* Total Profit
* Cost of Goods Sold (COGS)
* Units Sold

---

## 🎯 Objective

The goal of this project is to:

* Analyze financial performance across multiple dimensions
* Identify high-performing segments and products
* Track sales and profit trends over time
* Support data-driven business decisions

---

## 📌 Key Insights

* Government segment generates the highest revenue (~$13M)
* Amarilla and Paseo products contribute the highest profit share
* Sales peak in October, followed by a decline in November
* Germany and Canada lead in total sales performance

## 📈 Key Features

* Interactive filters (Country, Segment, Year)
* KPI cards for Sales, Profit, COGS
